# Hangman Game using Python

## Description
This is a simple text-based Hangman Game developed using Python.

The player guesses a hidden word one letter at a time. The player gets only 6 wrong chances to guess the word correctly.

## Features
- Random word selection
- 5 predefined words
- Maximum 6 wrong guesses
- Console-based game

## Concepts Used
- Random Module
- While Loop
- If-Else
- Strings
- Lists

## How to Run
1. Install Python
2. Open the project folder
3. Run `Hangman.py`
